# IDAI105-2505525--AashrutiAhuja
Analyzes Black Friday sales data to reveal customer behavior, trends, insights, and patterns


# Project Overview

This project applies data mining techniques to Black Friday sales data to understand purchasing behavior and generate meaningful insights for decision-making.


User Personas

1. Retail Manager

* Wants to understand customer buying patterns
* Uses dashboard to identify top-selling products and trends

2. Data Analyst

* Interested in exploring data through visualizations
* Uses clustering and correlation analysis for deeper insights

3. Marketing Executive

* Looks for product combinations and customer segments
* Uses association rules to create targeted promotions

4. Business Owner

* Focuses on high-level insights
* Uses dashboard to make strategic decisions


# User Flow

1. Login Page

   * User enters username and password (any input allowed)
   * Clicks login and enters dashboard

2. Dashboard

    User navigates through sidebar sections:

     * Dataset Overview → Understand data
     * Data Cleaning → View preprocessing
     * EDA → Explore patterns through graphs
     * Segmentation → View customer clusters
     * Association Rules → Discover product combinations
     * Anomaly Detection → Identify unusual behavior
     * Insights → Read summarized findings


# Features

Dataset Overview

* Displays dataset shape and sample data

Data Cleaning

* Handles missing values
* Encodes categorical variables

Exploratory Data Analysis (EDA)

* Purchase distribution
* Gender spending analysis
* Age trends
* Product category insights
* Correlation heatmap

Customer Segmentation

* K-Means clustering
* Groups customers by behavior

Association Rules

* Apriori algorithm
* Finds frequently bought products

Anomaly Detection

* Detects unusual purchases
* Identifies outliers

Insights

* Summarizes key findings


# Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib / Seaborn
* Scikit-learn
* Mlxtend
* Streamlit / Flask


# Conclusion

This project shows how data mining helps uncover valuable insights from retail data and supports better decision-making.
